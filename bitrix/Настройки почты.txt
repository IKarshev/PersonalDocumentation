Бывает, что сразу же после установки Веб-окружения 1С-Битрикс там не работает почта (даже если в меню Веб-окружения настройки почты заданы корректно).

Решается проблема просто - установкой sendmail
yum install sendmail
service sendmail start

Только нужно будет убедиться, что запуск sendmail добавился в автозагрузку..
Добавить sendmail в автозагрузку можно так:
chkconfig --add sendmail

	2.1 Включить доступ "С сервера imap.yandex.ru по протоколу IMAP" (Пароли приложений и OAuth-токены) https://www.dextra.ru/about/forclients/mail_configuration/
3. Настроить smtp через BitriVM:
4. Ввести почту через админку в главном модуле и в настройках сайта

SMTP можно указывать в (теперь вроде нельзя указать несколько smtp аккаунтов через ',') "Дополнительный параметр для передачи функции mail" — --{smtp-аккаунт} --{почта из настроек smtp}
===========================================================
# smtp account configuration for main.akademshop.ru
account main.akademshop.ru
logfile /home/bitrix/msmtp_main.akademshop.ru.log
host smtp.yandex.ru
port 587
from akademshop@yandex.ru
aliases /etc/aliases
keepbcc off
auth on
user akademshop@yandex.ru
password 3HP2F*p!fv.is!#

tls on
tls_certcheck off
tls_starttls on
===========================================================
